移动端触摸浮层阻止窗体滑动

Posted 2018-01-19

移动端触摸浮层，当滚动条到顶或者到底的时候 window 窗体通常会随着滑动。会导致退出浮层的时候不在页面原先的位置。

目前流行的 css 方法给 html 和 body 加 overflow: hidden; height: 100% 是无效的。正确的解决办法是给 body 加 position: fixed，可以有效阻止移动端的窗体滑动。但是因为窗体高度被改变的原因，一旦加上以上样式后，页面会回到最顶部。因此还要附加记录打开浮层前的 scrollTop 值，在关闭浮层后回到被记录 scrollTop 的位置。
